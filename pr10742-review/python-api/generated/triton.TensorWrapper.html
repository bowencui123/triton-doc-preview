triton.TensorWrapper

class triton.TensorWrapper(self, base, dtype)

Presents a tensor under a different element type.

Returned by triton.reinterpret(). The wrapper forwards the storage, shape, strides, and device of the wrapped tensor while reporting dtype as the new type, so a kernel reads the same bytes under a different interpretation.
