triton.compile

triton.compile(src, target=None, options=None, _env_vars=None)

Compiles a kernel to a device binary.

This is the lowering entry point used by triton.jit(); it can also be called directly for ahead-of-time compilation.

Parameters:

  • src – the kernel to compile, given either as an ASTSource (a @triton.jit function together with its signature) or as a path or string pointing at a textual IR module.

  • target – the GPUTarget to compile for; defaults to the target of the currently active device.

  • options – backend-specific compilation options (e.g. num_warps, num_stages) layered over the backend defaults.

Returns:

a compiled kernel that can be launched on the device.
