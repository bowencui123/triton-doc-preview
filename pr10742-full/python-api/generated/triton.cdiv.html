triton.cdiv

triton.cdiv()

Returns the ceiling division of x by y (that is, x / y rounded up to the nearest integer).

Commonly used on the host to size a launch grid, e.g. grid = (triton.cdiv(n_elements, BLOCK_SIZE), ).

Parameters:

  • x – the dividend.

  • y – the divisor.
