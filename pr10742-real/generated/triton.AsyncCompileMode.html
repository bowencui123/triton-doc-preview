triton.AsyncCompileMode

class triton.AsyncCompileMode(self, executor: Executor, *, ignore_errors=False)

A context manager that compiles kernels asynchronously on an executor.

Inside the with block, kernel compilations triggered by launches are submitted to the given concurrent.futures.Executor and run in the background, each returning a FutureKernel. Any compiles still outstanding are finalized when the block exits.
