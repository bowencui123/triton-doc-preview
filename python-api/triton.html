triton

GREEN highlights = NEW or CHANGED relative to upstream Triton (formula / example / Returns / params).  绿色 = 相比官方原版的新增或修改。 Un-annotated clean version is on the main branch.

jit

Decorator for JIT-compiling a function using the Triton compiler.

autotune

Decorator for auto-tuning a triton.jit'd function.

heuristics

Decorator for specifying how the values of certain meta-parameters may be computed.

Config

An object that represents a possible kernel configuration for the auto-tuner to try.

Compilation

compile

Compile a Triton kernel source into a runnable CompiledKernel.

AsyncCompileMode

Context manager that compiles Triton kernels asynchronously on an executor.

FutureKernel

A handle to a kernel whose compilation is running asynchronously.

JIT / Kernel

JITFunction

Callable wrapper produced by the triton.jit() decorator.

KernelInterface

Mixin defining the launch interface shared by all callable Triton kernels.

MockTensor

Can be used in place of real tensors when calling:

Tensor Reinterpretation

reinterpret

Reinterpret a tensor's storage as a different Triton dtype, without copying.

TensorWrapper

Lightweight view that reinterprets an existing tensor as a different dtype.

Memory Management

set_allocator

The allocator function is called during kernel launch for kernels that require additional global memory workspace.

Utilities

cdiv

next_power_of_2

Return the smallest power of 2 greater than or equal to n

constexpr_function

Wraps an arbitrary Python function so that it can be called at compile-time on constexpr arguments in a Triton function and returns a constexpr result.

aggregate

must_use_result

If the result of this function is unused, throw an error.

Exceptions

TritonError

Base class for all exceptions raised by Triton.

CompilationError

Base class for all errors raised during compilation

OutOfResources

Raised when a kernel requests more of a hardware resource than is available.

InterpreterError

Raised when an error occurs while running a kernel in the interpreter.
