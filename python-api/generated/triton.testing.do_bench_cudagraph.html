triton.testing.do_bench_cudagraph

GREEN highlights = NEW or CHANGED relative to upstream Triton (formula / example / Returns / params).  绿色 = 相比官方原版的新增或修改。 Un-annotated clean version is on the main branch.
triton.testing.do_bench_cudagraph(fn, rep=20, grad_to_none=None, quantiles=None, return_mode='mean')

Benchmark the runtime of the provided function.

Parameters:

  • fn (Callable) – Function to benchmark

  • rep (int) – Repetition time (in ms)

  • grad_to_none (torch.tensor, optional) – Reset the gradient of the provided tensor to None

  • return_mode (str) – The statistical measure to return. Options are “min”, “max”, “mean”, “median”, or “all”. Default is “mean”.

Returns:

The benchmarked runtime in milliseconds. A single float for the selected return_mode, or a list of floats if quantiles is provided or return_mode="all".

Return type:

float | list[float]

Example:

import torch
import triton

def fn():
    return x + y

x = torch.rand(2 ** 20, device="cuda")
y = torch.rand(2 ** 20, device="cuda")
ms = triton.testing.do_bench_cudagraph(fn, rep=100)
min_ms, med_ms, max_ms = triton.testing.do_bench_cudagraph(fn, quantiles=[0.2, 0.5, 0.8])
