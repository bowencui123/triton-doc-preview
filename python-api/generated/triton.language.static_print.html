triton.language.static_print

GREEN highlights = NEW or CHANGED relative to upstream Triton (formula / example / Returns / params).  绿色 = 相比官方原版的新增或修改。 Un-annotated clean version is on the main branch.
triton.language.static_print(*values, sep: str = ' ', end: str = '\n', file=None, flush=False, _semantic=None)

Print the values at compile time. The parameters are the same as the builtin print.

NOTE: Calling the Python builtin print is not the same as calling this, it instead maps to device_print, which has special requirements for the arguments.

tl.static_print(f"BLOCK_SIZE={BLOCK_SIZE}")
Parameters:

  • values – the compile-time values to print.

  • sep (str) – string inserted between values.

  • end (str) – string appended after the last value.

  • file – a file-like object to print to, or None for the default stream.

  • flush (bool) – whether to forcibly flush the stream.

Returns:

None
