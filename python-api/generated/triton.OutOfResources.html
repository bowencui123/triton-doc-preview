triton.OutOfResources

exception triton.OutOfResources(self, required, limit, name)

Raised when a kernel requests more of a hardware resource than is available.

Typically triggered when the requested shared memory, registers, or similar resource exceeds the hardware limit. Reducing block sizes or num_stages often resolves it. The offending resource is named by name, with the requested and available amounts in required and limit.
