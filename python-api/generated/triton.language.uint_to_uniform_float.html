triton.language.uint_to_uniform_float(x)

Numerically stable function to convert a random uint into a random float uniformly sampled in [0, 1).

Parameters:

x – A block of random integer bits (32-bit or 64-bit, signed or unsigned).

Returns:

A block of float32 uniformly sampled in \([0, 1)\), with the same shape as x.

Return type:

float32 tensor
