triton.constexpr_function

GREEN highlights = NEW or CHANGED relative to upstream Triton (formula / example / Returns / params).  绿色 = 相比官方原版的新增或修改。 Un-annotated clean version is on the main branch.
triton.constexpr_function(fn: T) → ConstexprFunction[T]

Wraps an arbitrary Python function so that it can be called at compile-time on constexpr arguments in a Triton function and returns a constexpr result.

Example:

@triton.constexpr_function
def double(n):
    return 2 * n

@triton.jit
def kernel(x_ptr, n, BLOCK: tl.constexpr):
    WIDE: tl.constexpr = double(BLOCK)
    offs = tl.arange(0, WIDE)
    mask = offs < n
    x = tl.load(x_ptr + offs, mask=mask)
GREEN = added vs upstream
